##  Getting Started

### Prerequisites

* Flutter SDK
* Android Studio / VS Code
* Android Emulator or Physical Device

### Installation

1. Clone the repository

```bash
git clone https://github.com/Harsh-Mandaliya/flutter-todo-app.git
```

2. Navigate to project directory

```bash
cd flutter-todo-app
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
